1 Fibonacci Sequence Function

1.1 Functionality

  • fib: Takes a non-negative integer n and returns the nth Fibonacci number.

1.2 Implementation

(defn fib [n]
  (nth (map first (iterate (fn [[a b]] [b (+' a b)]) [0 1])) n))

1.2.1 Examples:

Produces nth number in the fibonacci sequence:

(deftest fib-test
  (testing
      (is (= (sut/fib 0) 0)))
  (testing
      (is (= (sut/fib 12) 144)))
  (testing
      (is (= (sut/fib 69) 117669030460994))))
PASS: (= 0 0)PASS: (= 144 144)PASS: (= 117669030460994 117669030460994)

Negitive numbers are out of bounds.

(deftest fib-neg-n-out-of-range-test
  (is
   (try (sut/fib -1)
        (catch Exception e
          (= (type e) java.lang.IndexOutOfBoundsException)))))
PASS: true

1.3 Properties

1.3.1 Parity Invarient

Every third Fibonacci number is even. This is because an odd + even = odd so we get the cycle:

0+1=1 -> 1+1=2 -> 1+2=3 -> ...
E+O=O -> O+O=E -> O+E=O -> ...
(def parity-invariant
  (prop/for-all
   [v gen/pos-int]
   (let [fib-number (sut/fib v)]
     (if (zero? (rem v 3))
       (even? fib-number)
       (odd? fib-number)))))
(tc/quick-check 10000 parity-invariant)
{:result true,
 :pass? true,
 :num-tests 10000,
 :time-elapsed-ms 50,
 :seed 1771227759129}
